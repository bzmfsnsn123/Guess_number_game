# Guess-Number-Game
Python 入门猜数字小游戏，新手练手项目

## 项目介绍
经典控制台猜数字游戏，程序随机生成1~100的数字，玩家不断猜测，根据大小提示推理答案，统计猜测次数。

## 运行环境
Python3 环境，仅使用标准库 random，无需安装第三方包。

## 运行方式
1. 克隆仓库到本地
2. 运行 `guess_number.py`
3. 根据提示输入数字游玩

## 用到知识点
1. random.randint 随机数生成
2. while 循环 + 分支if判断
3. try-except 异常输入处理
4. 数值范围校验
